# TUI Project Manager

Local-first terminal project manager for markdown-based trackers.

This repo is based on the markdown tracker workflow from `/project-manager`.
That repository is the human-authored planning source, and this app evolves from
that structure so the origin stays traceable for future contributors and agents.
